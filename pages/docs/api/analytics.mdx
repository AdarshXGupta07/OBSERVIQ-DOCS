# Analytics API

All analytics endpoints require authentication. Data is scoped to your team only.

---

## `GET /analytics/overview`

Overall stats across all traces.

**Response:**
```json
{
  "total_calls": 1240,
  "avg_latency_ms": 342,
  "total_cost_usd": 2.401234,
  "success_rate": 98.4,
  "error_count": 20
}
```

---

## `GET /analytics/by-model`

Cost and performance breakdown per AI model.

**Response:**

```json
  {
    "model": "llama-3.3-70b-versatile",
    "call_count": 890,
    "avg_latency_ms": 420,
    "total_cost_usd": 1.820000,
    "success_rate": 99.1
  },
  {

    "avg_latency_ms": 180,
    "total_cost_usd": 0.580000,
    "success_rate": 96.8
  }
]
```
